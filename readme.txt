== Description ==

This plugin allows you to add one or more menu as tree using shortcodes.


See a full description here:

http://royaltechbd.com/royal-tree/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `royal-tree` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add shortcodes in your posts (ex: [royal_tree menu="family"]) here menu name is family

== Frequently Asked Questions ==
= Is there documentation for this plugin? =

http://royaltechbd.com/royal-tree/

= How do I add a menu as tree to my post =

Using shortcodes in the edit box for your post.  The address parameter for the address, and the "menu" parameter for menu

Ex:
`[royal_tree menu="family"]`
